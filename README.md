# Финансовый трекер

Веб-приложение для управления личными финансами: счета, транзакции, категории, бюджеты. С поддержкой аудита действий пользователей.

## Возможности

- **Управление счетами** – CRUD, пополнение, снятие.
- **Транзакции** – доходы/расходы с категориями, фильтрация, группировка по счетам.
- **Бюджеты** – установка лимитов на категории, отслеживание трат.
- **Аудит** – логирование всех операций (создание, изменение, удаление) с параметрами и результатами.
- **Асинхронное логирование** – не влияет на производительность.
- **Безопасность** – Spring Security, пароли зашифрованы BCrypt.

## Стек технологий

- Java 17
- Spring Boot 3.2
- Spring Data JPA + Hibernate
- Spring Security
- MySQL/PostgreSQL (в зависимости от версии проекта)
- Thymeleaf (для веб-интерфейса)
- Lombok
- MapStruct
- Jackson (JSON)
- Maven
